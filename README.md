# Hermes Proposals

Repository untuk tracking research dan ide dari Hermes Agent.

## Struktur

```
hermes-proposals/
├── README.md              # Index ini
├── PENDING/               # Proposal belum diapprove Iqbal-sama
├── APPROVED/              # Proposal yang sudah diapprove
└── REJECTED/              # Yang tidak dipilih
```

## Format Proposal

```markdown
# Proposal: [Judul]
Date: YYYY-MM-DD
Source: [Sumber research - Reddit, YouTube, Twitter, dll]

## Findings (Research Results)
- Finding 1
- Finding 2

## Proposed Actions
1. [Action 1] — Expected impact + ROI estimate
2. [Action 2] — Expected impact + ROI estimate

## Priority: High/Medium/Low
## Status: PENDING → [APPROVED/REJECTED]
```

## Workflow

1. **Hermes research** → generate proposal during overnight cron jobs
2. **Iqbal-sama review** → approve/reject/edit proposal
3. **Approved** → move to APPROVED/, implement
4. **Rejected** → move to REJECTED/, learning
